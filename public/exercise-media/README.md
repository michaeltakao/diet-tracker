Exercise demo images sourced from
[free-exercise-db](https://github.com/yuhonas/free-exercise-db) (The
Unlicense — public domain, no attribution required).

Each `<slug>.jpg` is the first frame (`images[0]`) of the matching
free-exercise-db entry, downloaded 2026-07-22. `<slug>` is derived from this
repo's own `nameEn` (see `lib/exercise-db.ts`), not free-exercise-db's
internal id — the mapping is name-exact-match only (52/206 curated
exercises); no fuzzy matching was used, to avoid attributing the wrong
exercise's form to a name that merely looks similar.
